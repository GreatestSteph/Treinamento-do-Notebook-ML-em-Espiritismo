## Mini Tutorial - Como utilizar um Notebook ML:
É muito simples, crie um notebook, insira suas fontes e livros, e estará pronta para o uso!
Faça perguntas sobre o assunto, crie resumos, slides e entre outras opções de forma mais facilmente e eficientemente!
